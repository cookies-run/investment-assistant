# 投资助手 — macOS 安装说明

## 系统要求

- macOS 10.13 或更高版本
- Apple Silicon (M1/M2/M3) 或 Intel 芯片

## 安装步骤

1. 下载 `投资助手_0.1.0_aarch64.dmg`
2. 双击打开 DMG 文件
3. 将「投资助手」图标拖拽到「应用程序」文件夹
4. 从「启动台」或「应用程序」文件夹中打开「投资助手」

> **首次打开提示**：macOS 可能会提示「无法验证开发者」。请前往「系统设置 → 隐私与安全性」，点击「仍要打开」。

## 使用说明

### 1. 配置飞书推送（可选）

- 打开应用后，点击右上角 **⚙️ 设置** 图标
- 在「通知设置」中粘贴飞书群机器人的 Webhook 地址
- 开启「启用飞书推送」开关

> 获取 Webhook：在飞书群设置 → 添加「自定义机器人」→ 复制 Webhook 地址

### 2. 添加持仓标的

- **股票持仓**：点击「股票持仓」→「新增个股」，输入股票代码、成本价、持有数量、止盈止损线
- **基金持仓**：点击「基金持仓」→「新增基金」，输入基金代码、持有份额、成本、策略阈值

### 3. 分组管理

- 在「股票持仓」或「基金持仓」页面，点击「管理分组」
- 可以新建分组、将标的拖拽归类、创建对照组

### 4. 盘口博弈验证

- 点击任意股票卡片进入详情页
- 查看「盘口博弈验证」面板：
  - **OBI**：订单簿不平衡度
  - **AR**：主动攻击度（外盘/内盘）
  - **MI**：筹码锁定匹配度
- 系统会根据 4 分支决策树给出博弈建议

### 5. 策略报警

- 盘中监控：股票/基金涨跌幅超过设置的 `daily_profit_line` / `daily_loss_line` 时触发报警
- 累计监控：连续多日累计收益超过 `cumulative_profit_line` 时触发报警
- 报警同时写入系统并推送到飞书（如已配置）

## 数据存储

- 数据库：`~/Library/Application Support/投资助手/stock_monitor.db`
- 日志文件：`~/Library/Application Support/投资助手/server.log`

## 卸载

1. 将「投资助手」从「应用程序」文件夹拖拽到废纸篓
2. 删除数据（可选）：`rm -rf ~/Library/Application Support/投资助手/`

## 技术架构

- **前端**：Vue 3 + Vite + Element Plus + Tauri（macOS 原生壳）
- **后端**：Go + Gin + GORM + SQLite（随 App 一起打包为 Sidecar）
- **数据源**：新浪财经（L1 行情）、东方财富（外盘/内盘）、腾讯财经（分钟线）
